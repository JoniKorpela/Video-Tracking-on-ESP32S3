Ohjelmiston kääntämiseksi ja flashaamiseksi tarvitset ESP-IDF:n. Aseta ennen kääntämistä Wi-Fi tukiasemasi SSID ja salasana tiedostoon /tracker/main/cred.h. Nyt voit siirtyä ESP-IDF-terminaalilla kansioon /tracker ja ajaa komennon "idf.py build flash monitor" Kun monitori aukeaa, ja ESP yhdistyy wifiin, huomioi monitorista minkä IP-osoitteen ESP saa reitittimeltä. Aseta tämä IP-osoite tiedostoon /tracker/client/client.py muuttujaan HOST. Nyt voit ajaa tiedoston client.py. Voit rajata seurattavan objektin klikkaamalla videota, keskeyttää seurannan painamalla näppäintä "t" ja sulkea ohjelman painamalla näppäintä "q".
