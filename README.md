Ohjelmiston kääntämiseksi ja flashaamiseksi tarvitset ESP-IDF:n. Aseta ennen kääntämistä Wi-Fi tukiasemasi SSID ja salasana tiedostoon /tracker/main/cred.h. Nyt voit siirtyä ESP-IDF-terminaalilla kansioon /tracker ja ajaa komennon "idf.py build flash monitor" Kun monitori aukeaa, ja ESP yhdistyy wifiin, huomioi monitorista minkä IP-osoitteen ESP saa reitittimeltä. Aseta tämä IP-osoite tiedostoon /tracker/client/client.py muuttujaan HOST. Nyt voit ajaa tiedoston client.py. Voit rajata seurattavan objektin klikkaamalla videota, keskeyttää seurannan painamalla näppäintä "t" ja sulkea ohjelman painamalla näppäintä "q".

To compile and flash the software, you need ESP-IDF. Before compiling, set your Wi-Fi access point’s SSID and password in the file /tracker/main/cred.h. You can then move to the /tracker directory in the ESP-IDF terminal and run the command "idf.py build flash monitor" When the monitor opens and the ESP connects to Wi-Fi, note the IP address that the ESP receives from the router in the monitor output. Set this IP address in the file /tracker/client/client.py in the variable HOST. You can now run the client.py file. You can select the object to be tracked by clicking on the video, stop tracking by pressing the "t" key, and close the program by pressing the "q" key.
